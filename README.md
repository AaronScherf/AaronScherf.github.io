# Personal & Academic Portfolio Website

This repository contains the source code and content for my personal academic portfolio website, hosted at [aaronscherf.github.io](https://aaronscherf.github.io/). 

The site is built using **[Hugo](https://gohugo.io/)** (Extended) and leverages an academic template design optimized for presenting research, data science projects, and professional experience.

## 🚀 Features

- **Biography & Core Interests:** An overview of academic background, professional experience, and research focus areas (including development economics, sustainable development, machine learning, and spatiotemporal data analysis).
- **Research & Publications:** A dedicated space for working papers, journal publications, and ongoing research projects.
- **Academic Timeline:** Comprehensive tracking of education, teaching assistantships, and professional milestones.
- **Projects & Open Source:** Showcase gallery for technical tools, data science applications, and coding implementations.
- **Blog & Notes:** A section for writing technical walk-throughs, mathematical notes, or research updates.

## 🛠️ Built With

- **[Hugo (Extended Version)](https://gohugo.io/)** - Fast, secure static site generator.
- **Tailwind CSS / Academic Theme Framework** - Modular, responsive design optimized for scholarly profiles.
- **[GitHub Pages](https://pages.github.com/)** - Free, reliable hosting directly from this repository.

## 💻 Local Development

To modify content, add publications, or preview changes locally before pushing them live, follow these steps:

### Prerequisites

Ensure you have the **extended version** of Hugo installed on your machine (required for processing advanced SCSS/CSS compilation):

- **macOS (Homebrew):** `brew install hugo`
- **Windows (Scoop):** `scoop install hugo-extended`
- **Linux (Ubuntu/Debian):** `sudo apt-get install hugo` (or download the extended `.deb` package from the Hugo GitHub releases page).
